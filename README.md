# Demo DBMS - Storage Strategies

Đây là chương trình Python mô phỏng và so sánh hiệu suất của các chiến lược lưu trữ (Storage Strategies) khác nhau trong Hệ quản trị Cơ sở dữ liệu (DBMS).

## Yêu cầu hệ thống
- Máy tính cài đặt **Python 3.x**. (Không yêu cầu thư viện bên ngoài vì chương trình chỉ sử dụng thư viện tiêu chuẩn `random`).

## Cách chạy chương trình
1. Mở Terminal (Command Prompt / PowerShell trên Windows hoặc Terminal trên macOS/Linux).
2. Di chuyển đến thư mục chứa file mã nguồn.
3. Chạy lệnh sau:
   ```bash
   python storage_demo.py
   ```
